# Contributing

Contributions are welcome.

## Local development

1. Create a Python virtual environment.
2. Install `requirements.txt`.
3. Run `run_web_app.ps1`.
4. Keep backend analysis code separate from frontend UI code.

## Before submitting

- Run the Python and frontend contract tests listed in `README.md`.
- Keep mobile controls at least 44px high.
- Do not add copyrighted or watermarked action images.
- Use project-local assets under `frontend/assets`.
- Preserve the fixed GitHub Pages and Render deployment paths.
- Avoid committing videos, virtual environments, model caches or tunnel binaries.
